Student Management System
A simple Java console application for managing student records.
Features

Add new students (ID, Name, Age)
Validate student information
Add marks for existing students
Calculate average marks and grades
Display all students
Search student by ID
Show student with the highest average mark
Menu-driven console interface

Classes

Student: Stores student data, marks, average, and grade calculations.
StudentManagementApp: Handles user input, menu options, and application flow.

How to Run

Open the project in your Java IDE.
Compile and run StudentManagementApp.java.
Use the menu to manage student records.

Technologies

Java
ArrayList
Object-Oriented Programming (OOP)
